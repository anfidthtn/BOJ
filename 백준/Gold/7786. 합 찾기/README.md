### 문제 설명

<p>두 자연수 l과 u가 주어졌을 때, l보다 크거나 같고, u보다 작거나 같은 모든 자연수의 자리수의 합을 구하는 프로그램을 작성하시오.</p>

### 입력 

 <p>첫째 줄에 l과 u이 주어진다. (0 ≤ l ≤ u ≤ 2,000,000,000)</p>

### 출력 

 <p>l과 r을 포함하여 그 사이에 있는 모든 수의 자리수의 합을 출력한다.</p>
